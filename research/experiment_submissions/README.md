| File | Phase | Notes |

| `submission_phase8b.csv` | Phase 8B (early reranker) | First pass at official-signal reranker |
| `submission_phase8b3.csv` | Phase 8B.3 | Headroom-scaled depth bonus; surface/trap penalties |

| `submission_phase8c3.csv` | Phase 8C.3 | Career evidence signal added |
| `submission_phase8c4.csv` | Phase 8C.4 | Production disclaimer penalty added; final ranking champion |
| `submission_phase8d1.csv` | Phase 8D.1 | Full-pool exposure audit (same candidates as C4) |

| `submission_phase8e1.csv` | Phase 8E.1 | Prose reasoning formatting (natural language) — pre-integration champion |
| `test_submission.csv` | Benchmark test run | Intermediate output from benchmark testing |

The **current final submission** is `submission.csv` at the repository root, generated by:

```bash
python -m backend.competition.rank \
  --candidates data/candidates.jsonl \
  --job data/job_description.docx \
  --output submission.csv
```
